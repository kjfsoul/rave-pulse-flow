# Project Tasks

Use this file to track tasks. Convert to Beads issues with: `./scripts/tasklist-to-beads.sh`

## Example Tasks

- [ ] Set up development environment
- [ ] Configure database connection
- [ ] Implement authentication
- [x] Create project structure

## Usage

1. Add tasks as markdown checklists: `- [ ] Task description`
2. Tell Cursor: "Process tasklist.md" or run `./scripts/tasklist-to-beads.sh`
3. Tasks are converted to Beads issues automatically
4. Check off items here as you complete Beads issues
